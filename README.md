EXP 9 : Study of Numpy Library

Theory
1. NumPy Array Creation

NumPy provides the array() function to create arrays from Python lists or tuples. A NumPy array is a collection of elements of the same data type stored in contiguous memory locations. Arrays allow faster mathematical operations compared to normal Python lists and are widely used in numerical computing.

2. np.zeros() Function

The np.zeros() function is used to create an array where all elements are initialized to zero. The size of the array is specified as a tuple representing rows and columns. This function is commonly used when initializing arrays before performing calculations.

3. np.ones() Function

The np.ones() function creates an array in which all elements are equal to one. It is useful for creating matrices or arrays with uniform values and is often used in mathematical and scientific computations.

4. np.eye() Function

The np.eye() function is used to generate an identity matrix. An identity matrix is a square matrix where all diagonal elements are 1 and all other elements are 0. Identity matrices are commonly used in linear algebra and matrix operations.

5. np.arange() Function

The np.arange() function generates a sequence of numbers within a specified range. It works similarly to Python’s range() function but returns the result as a NumPy array. It allows specifying the start value, stop value, and step size.

6. np.linspace() Function

The np.linspace() function generates evenly spaced numbers between two given values. Unlike arange(), this function creates a fixed number of elements within the specified interval.

7. Arithmetic Operations on Arrays

NumPy allows element-wise arithmetic operations such as addition, subtraction, multiplication, and division between arrays. These operations are performed efficiently due to NumPy’s vectorized computation capability.

8. np.mean() Function

The np.mean() function is used to calculate the average of all elements in an array. It is commonly used in statistical analysis and data processing.

9. np.median() Function

The np.median() function calculates the median value of an array. The median represents the middle value when the data elements are arranged in sorted order.

10. np.max() Function

The np.max() function returns the maximum value present in an array. It helps identify the largest element in a dataset.

11. np.min() Function

The np.min() function returns the minimum value present in an array. It is useful for finding the smallest element in a dataset.

12. np.sum() Function

The np.sum() function calculates the total sum of all elements in an array. It is commonly used in numerical and statistical calculations.

13. Determinant of Matrix (np.linalg.det())

The np.linalg.det() function is used to calculate the determinant of a square matrix. The determinant is a scalar value that plays an important role in solving linear equations and matrix algebra.

14. Transpose of Matrix (.T)

The transpose of a matrix is obtained by interchanging rows and columns. In NumPy, the transpose operation can be performed using the .T attribute.

15. Matrix Multiplication (np.dot() or @)

Matrix multiplication is an important operation in linear algebra. In NumPy, matrix multiplication can be performed using the np.dot() function or the @ operator between two compatible matrices.

The experiment demonstrated how NumPy simplifies numerical computations and improves efficiency in data processing.
